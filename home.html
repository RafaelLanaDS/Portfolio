HTML/CSS

Uso de Bootstrap, variáveis CSS, animações e hover effects. Desenvolvimento de temas de e-commerce do zero com layouts personalizados, banners em grid e boas práticas de organização de código.

JavaScript

JavaScript: Desenvolvimento de interfaces dinâmicas com JavaScript puro, manipulação do DOM, estruturas de controle, funções, arrays e objetos. Consumo de APIs RESTful, integração com Swiper.js para carrosséis responsivos e introdução ao React para construção de componentes modernos e reativos.

PHP

PHP: Desenvolvimento back-end com PHP no padrão MVC utilizando CodeIgniter. Criação de rotas, controllers e models, operações CRUD completas integradas ao MySQL e compreensão do ciclo de requisição e resposta HTTP com formulários e validações.

MySql

MySQL: Design e implementação de bancos de dados relacionais com MySQL. Escrita de consultas SQL com SELECT, WHERE, ORDER BY, GROUP BY e operações CRUD completas. Relacionamentos entre tabelas, normalização básica e integração com back-end via CodeIgniter.
